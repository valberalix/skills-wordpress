GitHub Meta Tag é um plugin que cria uma meta tag no head do documento, contendo a URL do seu perfil.

== Description ==

Adicione seu perfil do GitHub em uma meta tag a partir da página de configurações do seu WordPress.

> **Nota: Este plugin requer PHP 7.0 ou superior para ser ativado.**

[**Repositório no GitHub**](https://github.com/valberalix/skills-wordpress/)
