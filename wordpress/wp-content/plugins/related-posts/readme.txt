Related Posts é um plugin que cria um shortcode para listar posts de mesma categoria.

== Description ==

Liste posts da mesma categoria em suas publicações. Basta adicionar o shortcode ao seu post e pronto!

> **Nota: Este plugin requer PHP 7.0 ou superior para ser ativado.**

[**Repositório no GitHub**](https://github.com/valberalix/skills-wordpress/)
