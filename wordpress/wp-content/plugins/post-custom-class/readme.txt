Post Custom Class é um plugin que adiciona uma classe customizada a todas as tags "p" do seu post.

== Description ==

Customize seu texto facilmente com Post Custom Class. Ele adiciona uma classe customizada aos parágrafos do seu post, facilitando o uso de um css dedicado.

> **Nota: Este plugin requer PHP 7.0 ou superior para ser ativado.**

[**Repositório no GitHub**](https://github.com/valberalix/skills-wordpress/)
